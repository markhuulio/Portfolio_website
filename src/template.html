Selenium WebDriver

I decided to spend a day learning about website automation. Here is what I came up with, the working principle is quite simple:

1. Navigate to Jimm's website.
2. Find and select "Komponentit" category
3. Find and select "Näytönohjaimet" category
4. Sort items by price (High to low) from dropdown list
5. Select most expensive item and add it to basket
6. Navigate to basket and verify that basket is not empty


Link to github with video: https://github.com/markhuulio/RPA_tester

OpenCV

The color detection is made by binary tresholding which obviously isn't the best. I plan to move to HSV in time.

Self Balancing Robot

Early 2020 I decided to work on my programming and electronics skills. I started working on a self balancing robot that stabilises itself by sensing with gyro and accelerometers and applies corrective actions with stepper motors. This project included some quick 3D modeling, KiCAD layout to make the PCB and programming with arduino for stabilizing the robot.

Herb growing machine

This project was centered around making herbs last throughout the cold and dark finnish winter. It consists of a esp32, a pump for circulating water, ultrasonic distance sensor for sensing when water reservoir is nearly empty, lights, heat sinks and other supplementing electronics. Fun build even though I had trouble making the 3D print watertight as I wanted to keep the outer walls thin to save material.

Witcher coaster
